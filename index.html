📘 사용 설명서

1. 초기 환경 설정 (필수)

우측 상단의 톱니바퀴(⚙️)를 눌러 본인이 주로 테스트하는 기기 목록, 앱 버전 등을 저장하세요. 브라우저에 영구 반영됩니다.

2. 스마트 제목 Prefix

[현상 요약]칸에 핵심 현상만 적고 조건을 선택하면 제목이 규격에 맞춰 자동으로 조립됩니다.

3. 제목/본문 포맷팅

제목은 슬래시(/) 사이 공백 없이 밀도 있게, 본문은 공백을 포함하여 가독성 있게 자동 조절됩니다.

4. 업데이트 확인

문서 아이콘(📜)을 클릭하여 V1.0부터 현재까지의 전체 개발 히스토리를 상세히 확인하세요.
