📘 사용 설명서

1. 초기 환경 설정 (필수)

우측 상단의 톱니바퀴(⚙️)를 눌러 기기 목록과 버전을 저장하세요. 브라우저에 영구 반영됩니다.

2. 스마트 제목 Prefix

[현상 요약]칸에 핵심 현상만 적고 조건을 선택하면 제목이 자동으로 조립됩니다.

3. 개별 필드 CASE 프리셋

입력창 레이블 옆의 CASE +를 누르고 숫자를 선택하면 해당 칸에만 케이스 서식이 주입됩니다.

4. 업데이트 확인

문서 아이콘(📜)을 클릭하여 전체 개발 히스토리를 상세히 확인하세요.
