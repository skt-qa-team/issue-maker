📘 사용 설명서

1. 초기 환경 설정 (필수)

우측 상단의 톱니바퀴(⚙️)를 눌러 본인이 주로 테스트하는 기기 목록, 앱 버전 등을 한 번만 저장해 두세요.

2. 스마트 제목 Prefix

[현상 요약]칸에 핵심 현상만 적고 조건을 선택하면, 시스템이 규격화된 제목을 자동으로 조립해 줍니다. (맞춤법 자동 검사 지원)

3. 제목과 본문 분리

우측 결과창에서 조립된 제목과 본문을 개별적으로 복사하거나, 하단 버튼으로 한 번에 복사할 수 있습니다.

4. 업데이트 확인

우측 상단의 문서 아이콘(📜)을 클릭하시면 이 툴의 전체 업데이트 및 시스템 진화 내역을 확인할 수 있습니다.

  • 브라우저 내장 네이티브 맞춤법 검사 엔진(Spellcheck)을 활성화하여 자연어 작성 영역 오타 검출 지원
  • 소프트웨어 수정 이력을 체계적으로 확인할 수 있는 패치 노트(Changelog) 모달 탑재
V15.0

  • 출력 포맷의 구분자 양옆 공백 완전 제거 (예: STG/DEV/상용)로 문자열 밀도 최적화
  • 환경 설정 내 불필요한 기본 POC 및 기본 서버 항목을 제거하여 설정 모달 UI 복잡도 최소화
V14.0

  • 팀원 온보딩을 위한 좌측 사용 설명서 패널 너비 확장 (280px → 350px) 및 텍스트 가독성 개선
V13.0

  • 과거 버전 데이터를 자동 감지하여 끌어오는 데이터 마이그레이션(Data Migration) 방어 로직 구축
V12.0
