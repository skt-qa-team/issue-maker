📘 사용 설명서

1. 초기 환경 설정 (필수)

우측 상단의 톱니바퀴(⚙️) 아이콘을 눌러 본인이 주로 테스트하는 기기 목록, 앱 버전 등을 한 번만 저장해 두세요. 브라우저에 영구적으로 자동 반영됩니다.

2. 스마트 제목 Prefix

[현상 요약]칸에 결함의 핵심 현상만 적고 하단의 조건을 선택하면, 시스템이 [환경][OS][POC]... 형태의 완벽하게 규격화된 제목을 자동으로 조립해 줍니다.

3. 동적 폼 (Admin / PC)

POC 영역에서 Admin이나 PC Web을 선택하면 해당 리포트에 불필요한 OS, 기기 선택창이 알아서 숨겨지고 각 환경에 맞는 전용 URL 입력창이 나타납니다.

4. 데이터 초기화 및 복사

결함 티켓 작성이 끝난 후 우측 상단의 [🔄 새로 작성] 버튼을 누르시면, 미리 세팅해둔 환경 설정 값은 안전하게 유지된 채로 이번 리포트에 썼던 1회성 텍스트들만 깔끔하게 지워집니다.
