1. 환경/테마 설정

우측 상단의 버튼으로 기기 목록을 저장하거나 🎨(팔레트) 버튼을 눌러 나만의 가독성 테마를 설정하세요.

2. 개별 필드 CASE 프리셋

입력창 레이블 옆의 CASE +를 누르고 숫자를 선택하면 해당 칸에만 서식이 자동 주입됩니다.

3. 하이브리드 계정 연동

좌측 상단에서 접속 중인 팀원을 확인하세요. 로그인 시 구글 프로필 사진과 실명이 연동됩니다.

4. 업데이트 확인

📜 아이콘을 클릭하여 시스템의 진화 히스토리를 상세히 확인할 수 있습니다.
